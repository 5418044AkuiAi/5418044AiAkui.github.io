HTML要素の紹介

このwebサイトは日本大学文理学部情報科学科のWebプログラミングの演習課題である。 HTMLに様々ある要素の中から箇条書きを示すものについて3つ紹介する。

ol要素

箇条書きを示す際、順序を示すための数字やアルファベットを表示させたいときに 利用する要素がol要素である。<ol type="X">の形で使用することができ、Xの部分に 1・A・a・i・Ⅰのいずれかを入力することにより、順序の示し方を選択 することができる。

例

type="1"のとき

ソースコード

<p>目玉焼きの作り方</p>
<ol type="1" >
    <li>蓋つきのフライパン・玉子・キャベツ・油を用意する</li>
    <li>キャベツを千切りにする</li>
    <li>フライパンに油を引き、千切りキャベツを敷く</li>
    <li>キャベツの上に玉子を割る</li>
    <li>フライパンに蓋をし、蒸し焼く</li>
    <li>お好みのタイミングで完成</li>
</ol>
実行例

目玉焼きの作り方

  1. 蓋つきのフライパン・玉子・キャベツ・油を用意する
  2. キャベツを千切りにする
  3. フライパンに油を引き、千切りキャベツを敷く
  4. キャベツの上に玉子を割る
  5. フライパンに蓋をし、蒸し焼く
  6. お好みのタイミングで完成

type="A"のとき

ソースコード

<p>次の選択肢のうち印象派の作曲家を選びなさい</p>
<ol type="A" >
    <li>モーツァルト</li>
    <li>ベートーベン</li>
    <li>ドビュッシー</li>
    <li>ショパン<li>
</ol>

次の選択肢のうち印象派の作曲家を選びなさい

  1. モーツァルト
  2. ベートーベン
  3. ドビュッシー
  4. ショパン

ul要素

li要素を用いた時とは違い、データの順序を考慮せずに箇条書きをする際にul要素を用いる。

例

ソースコード

<p>鞄の中身</p>
<ul>
    <li>スマートフォン</li>
    <li>文具ポーチ<li>
    <li>筆箱</li>
    <li>PC</li>
    <li>iPad</li>
    <li>メモ用の裏紙</li>
    <li>イヤホン</li>
</ul>
実行例

鞄の中身

  • スマートフォン
  • 文具ポーチ
  • 筆箱
  • PC
  • iPad
  • メモ用の裏紙
  • イヤホン

li要素

箇条書きをする際、要素を判別するために利用するのがli要素である。箇条書きの項目それぞれを <li>X<li>と表現する。順序付きのリストを作る際、li要素にvalueの値を与える ことにより順序のラベルを変更する事ができる

例

ソースコード

<p>好きな音楽</p>
<ul>
    <li value = "3">diga diga doo</li>
    <li value = "1">mambo influenciade<li>
    <li value = "5">just a little bit</li>
    <li value = "2">color of the wind</li>
    <li value = "7">恋の激ダサエクスタシー</li>
    <li value = "4">Black Cherry</li>
    <li value = "6">アリバイ</li>
</ul>
実行例

好きな音楽

  1. diga diga doo
  2. mambo influenciade
  3. just a little bit
  4. color of the wind
  5. 恋の激ダサエクスタシー
  6. Black Cherry
  7. アリバイ
